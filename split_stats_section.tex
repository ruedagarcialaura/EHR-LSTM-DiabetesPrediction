
\subsection{Cohort Splits}
We report the number of patients used for training, validation, and testing after class balancing and patient-level splitting. The training set contains 714 patients (T2D: 357, Non-T2D: 357), the validation set contains 153 patients (T2D: 79, Non-T2D: 74), and the test set contains 153 patients (T2D: 74, Non-T2D: 79). In total, the balanced cohort used for splitting includes 1020 patients (T2D: 510, Non-T2D: 510). Stratified splitting was applied to preserve label proportions across splits.
